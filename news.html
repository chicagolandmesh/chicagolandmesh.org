Welcome to the New Chicagoland Mesh Website!
January 2025

We're excited to launch our redesigned website built with Bootstrap 5! The new site features improved accessibility, mobile responsiveness, and a clean design that works great in both light and dark modes.

Key improvements include:

  • Modern, responsive design that works on all devices
  • Automatic light/dark theme switching
  • Improved navigation and user experience
  • Better accessibility features
  • Faster loading times with static hosting

The website is now hosted on GitHub Pages and built with static HTML, CSS, and JavaScript - no build process required!

Community Growth Milestone
December 2024

Our Discord community has reached over 600 members! This incredible growth shows the strong interest in mesh networking throughout the Chicagoland area.

We've seen increased activity in:

  • Node deployments across the metro area
  • Technical discussions and troubleshooting
  • Hardware recommendations and reviews
  • Range testing and optimization

Network Expansion Updates
November 2024

The Chicagoland mesh network continues to expand with new nodes being deployed throughout the region. We now have coverage in downtown Chicago and several suburban areas.

Recent additions include:

  • High-elevation nodes in downtown Chicago
  • Suburban repeater stations
  • Mobile nodes for emergency communications
  • Solar-powered remote installations

Check out our network map to see current coverage areas and plan your node placement.
