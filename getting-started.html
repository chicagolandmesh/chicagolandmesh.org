Getting Started

Important: Mistakes to Avoid

Before you start, please read these common pitfalls to save yourself time and frustration:

  • Don't skip antenna selection: A proper antenna is crucial for good range and performance
  • Don't use the default channel: Configure your own channel settings for better security
  • Don't place nodes indoors only: Higher placement with line-of-sight improves range significantly
  • Don't ignore power management: Configure sleep modes for battery-powered nodes
  • Don't forget to test: Always test your setup before permanent installation

5 Install the Mobile App

Download the official Meshtastic app to configure and communicate with your device.

iOS App Android App
6 Configure Your Node

Set up your node with appropriate settings for the Chicagoland area.

Recommended Settings:
  • Region: US (915MHz)
  • Channel: Create a custom channel or use community channels
  • Node Name: Choose a unique, descriptive name
  • GPS: Enable if you want location sharing
  • Power: Adjust based on your power source
7 Test Your Setup

Verify everything is working before permanent installation.

Basic Tests:
  • Device powers on and connects to app
  • Can send messages to yourself
  • GPS location updates (if enabled)
  • Battery life meets expectations
Range Tests:
  • Test at different distances
  • Try various antenna orientations
  • Check performance indoors vs outdoors
  • Monitor signal strength (SNR/RSSI)
